# Titanic Data Cleaning & Visualization Project

## Objective
Clean and analyze the Titanic dataset using Python.

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Tasks Performed
- Handled missing values
- Removed Cabin column
- Checked duplicates
- Detected outliers
- Created visualizations
- Generated insights

## Key Findings
- Most passengers belonged to the 3rd class.
- More passengers died than survived.
- Most passengers were aged between 20 and 40 years.
- Fare and Passenger Class showed a strong negative correlation.

- Titanic_Data_Cleaning_Visualization.ipynb
- Titanic-Dataset.csv
